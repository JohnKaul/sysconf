# sysconf

## BRIEF

A utility to make changes to key/value type system configuration
files.

## DESCRIPTION


## DOWNLOAD INSTRUCTIONS

To clone a copy:

```bash
$ cd ~/<place>/<you>/<keep>/<your>/<code>
$ git clone git@git.local:server/sysconf.git
```

## BUILD INSTRUCTIONS 

```bash
$ cd sysconf 
$ make 
```
